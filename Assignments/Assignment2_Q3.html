a. Create a new tibble called CASchools_A2_3 (with twice as many rows as the original) called which contains the following variables from the original CASchools : school and county as well as two new columns: a single column containing both the reading AND math average exam scores and another column which indicates whether the ﬁrst column contains a read score or a math score. Name this new tibble CA_Schools_A2_3_LONG . Hint: The suggested name for the tibble gives away the answer on what to do!

CASchools_A2_3_LONG <- as_tibble(CASchools) %>% 
  select(school, county, read, math) %>% 
  pivot_longer(cols=c(read, math), names_to = "ScoreType", values_to = "Score")

b. Using CA_Schools_A2_3_LONG , create a summary table called CA_Schools_3_summary with a row for read and a row for math for each county where the columns are the minimum, maximum, median and IQR of the school average scores by county for that test.

CA_Schools_3_summary <- CASchools_A2_3_LONG %>% 
  group_by(county, ScoreType) %>% 
  select(county, ScoreType, Score) %>% 
  summarise_all(list(Min=min, Max=max, Med=median, IQR=IQR)) %>% 
  ungroup()
